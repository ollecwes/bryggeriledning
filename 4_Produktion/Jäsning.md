# SOP: Jäsning

**Syfte:** Att omvandla socker till alkohol och koldioxid samt skapa ölets smakprofil.

## 1. Pitching (Tillsätta jäst)
*   [ ] Kontrollera att vörten har rätt temperatur i tanken.
*   [ ] Tillsätt jäst (torrjäst eller skördad slurry) i rätt mängd enligt recept.
*   [ ] Stäng tanken (eller sätt på jäsrör).

## 2. Jäsningsförloppet
*   **Lagfas (0-24h):** Jästen förökar sig. Ingen aktivitet syns.
*   **Stormjäsning (Dag 2-5):** Kraftig aktivitet. Skum (Kräusen) bildas. Temperaturkontroll är kritisk här! Jästen genererar värme.
*   **Utjäsning:** Aktiviteten avtar. Jästen börjar flockulera (sjunka till botten).

## 3. Daglig kontroll
*   [ ] Mät temperatur (morgon/kväll).
*   [ ] Kontrollera tryck (om trycktank används).
*   [ ] Smaka av (sensorisk kontroll) för att upptäcka problem tidigt.

## 4. Diacetylrast (Viktigt för Lager)
När jäsningen närmar sig slutet (ca 75% utjäst), höj temperaturen 2–3 grader i 2 dagar. Detta hjälper jästen att städa upp diacetyl (smörkola).

## 5. Cold Crash
När FG (Final Gravity) är nått och stabilt i 2 dagar:
*   [ ] Sänk temperaturen till 0–4°C.
*   [ ] Låt stå i minst 2 dagar för att klarna ölet.
*   [ ] Skörda/dumpa jäst från konen.
