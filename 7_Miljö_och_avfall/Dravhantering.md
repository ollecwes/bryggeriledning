# Rutin: Dravhantering

**Syfte:** Att hantera restprodukten från mäskningen (drav) på ett miljösmart sätt.

## 1. Vad är drav?
Drav är krossad malt som lakats ur. Det består mest av skal och proteiner. Det är ett utmärkt djurfoder.

## 2. Hantering
*   **Uppsamling:** Dravet skottas/pumpas ur mäskkärlet till täta kärl (t.ex. pallboxar eller släpvagn).
*   **Hygien:** Drav surnar snabbt (särskilt på sommaren). Måste hämtas inom 24–48 timmar.
*   **Rengöring:** Kärlen måste tvättas ur noga efter varje tömning för att undvika flugor och lukt.

## 3. Avsättning (Lantbrukare)
Vi samarbetar med lokal lantbrukare [Namn på gård] som hämtar dravet.
*   **Avtal:** Muntligt eller skriftligt avtal om hämtningsfrekvens.
*   **Status:** Dravet klassas som foder, ej avfall, så länge det hanteras hygieniskt.

## 4. Alternativ
Om lantbrukaren inte kan hämta:
*   Kompostering (kräver tillstånd om stor skala).
*   Biogasanläggning (kräver transport).
