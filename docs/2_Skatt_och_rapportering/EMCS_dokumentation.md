# Process: EMCS-dokumentation

**Syfte:** Att säkerställa spårbarhet och skattekontroll vid flytt av alkoholvaror under skatteuppskov (duty suspension).

**Ansvarig:** Logistikansvarig

## 1. Vad är EMCS?
EMCS (Excise Movement and Control System) är ett EU-gemensamt elektroniskt system för att övervaka flytt av punktskattepliktiga varor.

## 2. När ska EMCS användas?
*   Vid försäljning till **Systembolaget** (varorna flyttas från vårt skatteupplag till Systembolagets skatteupplag).
*   Vid export till annat EU-land.
*   Vid flytt till annan godkänd upplagshavare i Sverige.

*> [!NOTE]*
*> Vid direktförsäljning till restaurang (som inte är upplagshavare) ska skatten betalas direkt, och EMCS används **inte**. Då ska istället ett förenklat ledsagardokument eller faktura medfölja, och skatten tas upp i nästa deklaration.*

## 3. Skapa ett e-AD (Elektroniskt administrativt dokument)
Innan transporten lämnar bryggeriet:
1.  Logga in i **EMCS** via Skatteverkets hemsida.
2.  Skapa ett nytt **e-AD**.
3.  Ange:
    *   Avsändare (vårt upplagsnummer).
    *   Mottagare (mottagarens punktskattenummer).
    *   Transportuppgifter (tid, transportsätt).
    *   Varubeskrivning (varukod för öl: 2203, mängd, alkoholhalt).
4.  Skicka in. Systemet genererar en **ARC-kod** (Administrative Reference Code).

## 4. Under transport
*   ARC-koden ska medfölja transporten (utskrivet e-AD eller koden tydligt angiven på fraktsedeln).

## 5. Kvittens
När mottagaren tar emot varan kvitterar de i EMCS.
*   **Kontroll:** Bevaka att kvittens ("Rapport om mottagande") inkommer inom 5 dagar efter leverans.
*   Om kvittens saknas: Kontakta mottagaren omedelbart. Utan kvittens kan vi bli skyldiga att betala skatten.
