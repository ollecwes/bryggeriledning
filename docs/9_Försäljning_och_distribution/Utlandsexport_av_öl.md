# Utlandsexport av öl

**Syfte:** Rutiner och krav för export av öl till utländska marknader

**Ansvarig:** Logistikansvarig

## 1. Bakgrund

Export av öl till utländska marknader är en viktig del av bryggeriers tillväxtstrategi och möjliggör spridning av svenska hantverkstraditioner internationellt. Export kräver förståelse för regelverk, logistik, tullhantering och dokumentationskrav som varierar mellan olika länder och marknader.

Detta dokument beskriver rutiner och krav för export av öl från Brygd Brewery till utländska marknader, med särskilt fokus på EU-export samt export till tredjeland.

## 2. Omfattning

Detta dokument omfattar:

* Export av färdigförpackad öl till privatimportörer och distributörer
* Export inom EU (intra-community trade)
* Export till tredjeland (utanför EU)
* Dokumentationskrav och certifiering
* Tull- och skattehantering
* Transportlogistik och spårbarhet

**Exkluderas:**
* Export av råvaror eller bryggeriutrustning
* Försäljning via e-handel direkt till konsument (regleras separat)

## 4. Procedur/Process

### 4.1 Förberedelse

#### 4.1.1 Marknadsanalys och godkännande
- [ ] Kartlägg potentiell marknad och efterfrågan
- [ ] Identifiera distributörer eller importörer
- [ ] Undersök konkurrensläge och prissättning
- [ ] VD godkänner expansion till ny marknad

#### 4.1.2 Regelverksanalys
- [ ] Kontrollera importlandets alkohollagstiftning
- [ ] Identifiera krav på etikettering (språk, varningar, innehållsförteckning)
- [ ] Undersök krav på certifikat (hälsointyg, ursprungsintyg, etc.)
- [ ] Kontrollera om produkten behöver registreras hos myndigheter
- [ ] Ta reda på gällande tull- och skattenivåer

**Viktiga resurser:**
* EU:s alkoholdirektiv och märkningsregler
* Destination country's customs website
* Business Sweden för marknadsinformation
* Livsmedelsverket för exportregler

#### 4.1.3 Certifiering och registrering
Beroende på destination kan följande krävs:

* **Hälsointyg** - utfärdas av Livsmedelsverket för vissa länder
* **Ursprungsintyg** - bekräftar svensk ursprung, utfärdas av Handelskammaren
* **Analysprotokoll** - alkoholhalt, innehållsförteckning
* **Registrering** - vissa länder kräver produktregistrering innan import

#### 4.1.4 Anpassning av etikett
- [ ] Översätt etikett till destinationslandets språk (om krav finns)
- [ ] Lägg till obligatoriska varningstexter
- [ ] Säkerställ att alkoholhalt anges enligt lokala regler
- [ ] Inkludera importörens kontaktuppgifter
- [ ] Få etikett godkänd av importör/distributör

### 4.2 Genomförande

#### 4.2.1 Orderhantering
1. **Motta order** från distributör/importör
2. **Bekräfta order** inkl. produktspecifikation, kvantitet, leveransdatum
3. **Kontrollera lagerstatus** och produktionsmöjlighet
4. **Reservera varor** i lagersystemet
5. **Bekräfta slutlig leverans** med transportdatum

#### 4.2.2 Dokumentförberedelse

**För EU-export:**
* Handelsfaktura (commercial invoice)
* Fraktsedel/CMR (transport document)
* Följesedel med partinummer för spårbarhet
* Packlista med antal kollin, vikt, innehåll

**För export till tredjeland, lägg till:**
* Tulldeklaration (exportanmälan i EMCS eller Tullverket)
* Ursprungsintyg (Certificate of Origin)
* Hälsointyg vid behov
* Proformafaktura vid behov

#### 4.2.3 Fysisk förberedelse
- [ ] Plocka produkter enligt packlista
- [ ] Kontrollera att rätt etikett används
- [ ] Kvalitetskontroll - kontrollera produktens skick
- [ ] Packa enligt transportkrav (pallning, stretchfilm, kantskydd)
- [ ] Märk kollin med destinations-, hanterings- och varningsetiketter
- [ ] Säkerställ temperaturkontroll vid behov

#### 4.2.4 Transport och spedition
1. **Välj transportör** - använd godkänd speditör med erfarenhet av alkoholtransporter
2. **Boka transport** - ange särskilda krav (temperatur, tidsram)
3. **Överlämna gods** tillsammans med alla dokument
4. **Meddela mottagare** om fraktnummer och beräknad ankomst

**Vanliga transportalternativ:**
* Lastbil (inom Europa)
* Kombinerad transport (lastbil + färja)
* Sjöfrakt (större volymer, längre avstånd)

#### 4.2.5 Tull- och skattehantering

**Export från Sverige:**
* Alkoholskatt betalas **inte** vid export
* Registrera export i Tullverkets system
* Bevara exportdokumentation i minst 7 år

**Import i destinationsland:**
* Mottagaren ansvarar för tull och importskatt
* Vid DDP (Delivered Duty Paid) betalar avsändaren - specificera i avtal
* Säkerställ att mottagare har rätt importlicenser

### 4.3 Uppföljning

#### 4.3.1 Leveranskontroll
- [ ] Bekräfta att gods har anlänt i god kondition
- [ ] Kontrollera eventuella transportskador
- [ ] Samla in kvitto/mottagningsbevis
- [ ] Arkivera alla transportdokument

#### 4.3.2 Betalning
- [ ] Fakturera enligt avtalade betalningsvillkor
- [ ] Följ upp betalning
- [ ] Hantera eventuella valutakursdifferenser

#### 4.3.3 Kundrelationer
- [ ] Samla feedback från importör/distributör
- [ ] Utvärdera försäljning på marknaden
- [ ] Planera för framtida beställningar
- [ ] Uppdatera produktinformation vid behov

#### 4.3.4 Avvikelsehantering
Vid problem (förseningar, skador, felaktiga dokument):
1. Dokumentera avvikelsen omedelbart
2. Kommunicera med transportör och mottagare
3. Utreda orsak och ansvar
4. Implementera korrigerande åtgärder
5. Uppdatera rutiner vid behov

## 5. Dokumentation

### 5.1 Obligatoriska dokument som ska arkiveras

| Dokument | Lagringstid | Ansvarig |
|----------|-------------|----------|
| Exportfakturor | 7 år | Ekonomiansvarig |
| Tulldeklarationer | 7 år | Ekonomiansvarig |
| Ursprungsintyg | 5 år | Logistikansvarig |
| Fraktsedlar (CMR) | 5 år | Logistikansvarig |
| Hälsointyg | 3 år | Kvalitetsansvarig |
| Avtal med distributörer | Under avtalstid + 3 år | VD |

### 5.2 Löpande dokumentation

* **Exportlogg** - översikt av alla exporter (datum, destination, volym, mottagare)
* **Kundregister** - kontaktuppgifter till importörer/distributörer
* **Marknadsspecifika krav** - dokument med krav per land
* **Incidentrapporter** - avvikelser vid export
