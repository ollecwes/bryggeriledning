# Rutin: Restaurangavtal

**Syfte:** Att bygga långsiktiga relationer med restaurangkunder och säkra betalning.

## 1. Ny kund
*   **Kreditkontroll:** Innan första leverans mot faktura, gör en kreditkoll (t.ex. via UC eller Ratsit).
    *   *Godkänd:* 30 dagar netto (standard).
    *   *Ej godkänd:* Förskottsbetalning eller kortbetalning vid leverans.
*   **Avtal:** Skriv gärna ett enkelt avtal om förväntad volym, leveransdagar och pantrutiner.

## 2. Order
*   Order tas emot via mail, telefon eller säljsystem.
*   Bekräfta alltid ordern och leveransdag till kunden.

## 3. Fatpant
Vi debiterar pant för stålfat (t.ex. 500 kr).
*   Panten krediteras när det tomma fatet lämnas tillbaka.
*   Håll koll på "fatsaldot" per kund för att undvika svinn.

## 4. Utrustning
Om vi lånar ut kylar eller tappkranar till kunden ska ett "Låneavtal för utrustning" skrivas under. Utrustningen tillhör bryggeriet.
