# Rutin: Skyddsutrustning (PPE)

**Syfte:** Att skydda personalen mot omedelbara risker.

## 1. Kravlista
Följande utrustning ska finnas och användas:

| Moment | Utrustning | Varför? |
| :--- | :--- | :--- |
| **Hantering av kemikalier (koncentrat)** | heltäckande skyddsglasögon/visir, kemhandskar (långa), förkläde. | Frätskador på ögon och hud. |
| **Hantering av malt (Krossning)** | Andningsskydd (P2/P3-mask). | Maltdamm är allergiframkallande ("bagarlunga"). |
| **Hett vatten / Vört** | Värmetåliga handskar, stövlar (byxor utanpå stövelskaftet!). | Brännskador. |
| **Flasktappning** | Skyddsglasögon. | Risk för glassplitter vid sprängning. |
| **Bullrande maskiner** | Hörselskydd (kåpor/proppar). | Hörselskador. |

## 2. Skötsel
*   Byt ut trasiga handskar omedelbart.
*   Rengör skyddsglasögon regelbundet.
*   Förvara utrustningen på anvisad plats (t.ex. "PPE-tavla").
