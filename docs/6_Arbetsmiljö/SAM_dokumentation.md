# Rutin: SAM-dokumentation (Systematiskt Arbetsmiljöarbete)

**Syfte:** Att undersöka, riskbedöma, åtgärda och följa upp arbetsmiljön för att ingen ska bli sjuk eller skada sig.
**Lagkrav:** AFS 2001:1.

## 1. Årlig genomgång (Skyddsrond)
En gång per år genomförs en fysisk skyddsrond där VD och skyddsombud (eller personalrepresentant) går igenom lokalen.
*   **Checklista:** Vi använder checklistan "Skyddsrond Bryggeri".
*   **Områden:** Maskinsäkerhet, kemikalier, buller, lyft, belysning, ventilation, brandskydd.

## 2. Riskbedömning
Vid förändringar (ny utrustning, ombyggnad) eller upptäckta risker görs en skriftlig riskbedömning.
1.  **Risk:** Vad kan hända? (T.ex. "Klämrisk vid krossen").
2.  **Sannolikhet/Konsekvens:** Hur troligt? Hur allvarligt?
3.  **Åtgärd:** Vad gör vi? (T.ex. "Montera skyddsgaller").
4.  **Ansvarig/Klart datum:** Vem fixar det?

## 3. Handlingsplan
Risker som inte kan åtgärdas direkt skrivs upp i en handlingsplan. Denna följs upp vid personalmöten.

## 4. Tillbud och Olyckor
Alla tillbud ("nästan-olyckor") och olyckor ska rapporteras.
*   **Syfte:** Att lära sig och förebygga att det händer igen.
*   **Allvarliga olyckor:** Ska anmälas till Arbetsmiljöverket och Försäkringskassan (inom 24h).
