# Rutin: Kemikalieförteckning

**Syfte:** Att ha kontroll över farliga ämnen och säkerställa rätt hantering.

## 1. Förteckning
En uppdaterad lista över alla kemikalier finns i pärmen "Kemikalier" samt digitalt.
Listan innehåller:
*   Produktnamn
*   Användningsområde
*   Farosymboler
*   Länk till Säkerhetsdatablad (SDB)

### Vanliga kemikalier i bryggeriet
*   **Lut (NaOH):** Starkt frätande bas. För rengöring (CIP).
*   **Syra (Fosforsyra/Salpetersyra):** Starkt frätande. För passivisering/stenborttagning.
*   **Perättiksyra (PAA):** Starkt oxiderande/frätande. För desinfektion.
*   **Glykol:** Kylmedium (giftigt vid förtäring om ej livsmedelsklassad).

## 2. Säkerhetsdatablad (SDB)
Leverantören ska bifoga SDB vid leverans.
*   Dessa ska finnas tillgängliga där kemikalierna förvaras (t.ex. i städskrubb/kemrum).
*   Läs SDB *innan* du använder en ny produkt!

## 3. Förvaring
*   Kemikalier ska förvaras invallade (så att spill inte når avlopp).
*   Syra och Bas får **ALDRIG** förvaras så att de kan blandas vid läckage (kraftig värmeutveckling/explosion).
*   Ska vara inlåsta eller oåtkomliga för obehöriga.

## 4. Nödläge
*   **Ögonspolning:** Ska finnas lättillgängligt. Spola i minst 15 minuter vid stänk. Åk till sjukhus.
*   **Nöddusch:** Ska finnas i anslutning till kemhantering.
*   **Spill:** Använd absorptionsmedel (Absol) vid mindre spill. Spola med mycket vatten (om det kan ske säkert).
