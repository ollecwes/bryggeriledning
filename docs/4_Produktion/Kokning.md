# SOP: Kokning (CCP 1)

**Syfte:** Sterilisera vörten, isomerisera humle (beska), koagulera proteiner och driva bort DMS.

**Kritisk Styrpunkt (CCP 1):** Eliminering av mikroorganismer.

## 1. Uppkok
*   [ ] Starta värmen så snart botten är täckt i kokkärlet.
*   [ ] Vakta uppkoket noga för att undvika överkok ("boil over"). Stäng av värmen eller spraya med vatten om skummet stiger för fort.

## 2. Kokprocessen
*   **Tid:** Minst 60 minuter rullande kok.
*   **Lock:** Ska vara ÖPPET eller ha ångskorsten för att DMS (dimetylsulfid, smakar kokta grönsaker) ska kunna ångas bort.
*   **Humlegivor:** Tillsätts enligt recept:
    *   *60 min:* Bitterhumle.
    *   *15-20 min:* Smakhumle + eventuellt klarningsmedel (Protafloc/Irish Moss).
    *   *0-5 min:* Aromhumle.

## 3. Whirlpool
*   [ ] Stäng av värmen.
*   [ ] Starta whirlpool (rotera vörten) i 10–15 minuter.
*   [ ] Låt stå stilla i 15–20 minuter så att "druv" (humlerester och proteiner) lägger sig som en kon i mitten.

## 4. Kylning (CCP 2)
*   [ ] Pumpa vörten genom plattvärmeväxlaren till jästanken.
*   [ ] Syresätt vörten på vägen (om jäststammen kräver det).
*   **Krav:** Vörten ska nå måltemperatur (t.ex. 18°C) inom 60 minuter för att undvika bakterietillväxt.
