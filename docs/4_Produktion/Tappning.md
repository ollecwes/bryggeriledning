# SOP: Tappning

**Syfte:** Att förpacka ölet för försäljning med bibehållen kvalitet.

**Kritisk Styrpunkt (CCP 3 & 4):** Flasksköljning och Märkning.

## 1. Förberedelser
*   [ ] Rengör och desinficera tapplinan (CIP).
*   [ ] Kontrollera att rätt etiketter, kapsyler och flaskor/fat finns framme.
*   [ ] Kontrollera ölets CO2-halt och temperatur.

## 2. Flasktappning
1.  **Sköljning (CCP 3):** Alla flaskor ska sköljas ur med vatten/luft för att avlägsna damm och ev. glasflis.
2.  **Fyllning:** Fyll med mottryck för att undvika skumning och syresättning.
    *   *Syre:* Syre är fienden! Minimera syrekontakt ("Low Dissolved Oxygen" - DO).
3.  **Kapsylering:** Kapsylen ska sitta tätt. Kontrollera "crimpningen" regelbundet.
4.  **Sköljning:** Spola av flaskan utvändigt (ta bort ölrester som kan mögla).
5.  **Märkning (CCP 4):** Applicera etikett.
    *   *Kontroll:* Stämmer etiketten med ölet? Är Bäst-före-datum och Batchnummer läsbart?

## 3. Fattappning
1.  **Rengöring:** Faten ska vara rengjorda med het lut och syra (helst i fatdiskmaskin).
2.  **Fyllning:** Fyll via fatkopplingen (ut-porten) för att fylla från botten. Pys ut övertryck via in-porten.
3.  **Vägning:** Fyll till rätt vikt (t.ex. 30 liter + fatvikt). Överfyll inte!

## 4. Efterkontroll
*   Låt flaskorna stå i rumstemperatur 1 dygn för att se att inget läcker, ställ sedan i kyl.
